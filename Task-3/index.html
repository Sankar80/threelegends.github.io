Front End Language
The main Front-End languages used in web development
HTML,CSS and JavaScript
HTML
The Hyper Text Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
CSS
CSS stands for Cascading Style Sheets.CSS describes how HTML elements are to be displayed on screen.
JAVA SCRIPT
JavaScript is a dynamic programming language thats used for web development,in web applications,for game development,and lots more.




BOOTSTRAP
BOOTSTRAP is a free, open source front-end development framework for the creation of websites and web apps.
REACT
REACT is a free, front-end JavaScript library for building user interfaces based on components.
ANGULAR
Angular is a platform and framwork for building single page client applications using HTML and TypeScript.
